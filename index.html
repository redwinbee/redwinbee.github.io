I am currently an undergraduate student at Queens College, CUNY pursuing a Computer Science, B.S degree with a personal preference on compilers, interpreters, and regular expressions. I am currently working on stix which is the bytecode implementation (clox) from the Crafting Interpreters book. The goal of this project is to learn about the techniques used in implementing compiled programs that run on top of a VM such as bytecode representation, static analysis, and garbage collection.

As you can tell from this website, I am most definitely not a web designer. Something about creating UI elements and crafting the perfect UX experience was something that never clicked with me—perhaps because many of the websites that I frequent run incredibly slow on my machine; despite running relatively new hardware. As such, you'll find that this website was created using plain HTML/CSS and should load very quickly on almost any machine.

Call it old school, but this is how I was taught to write websites.
